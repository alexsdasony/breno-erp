## 🛠️ Stack Tecnológica

### Frontend
- **React** + **Vite**
- **Tailwind CSS**
- **Framer Motion**
- **Radix UI**
- **Supabase Client** (Autenticação e Banco)

### Backend
- **Node.js** + **Express**
- **Supabase** (Banco de dados e autenticação)
- **JWT Authentication** (Custom)
- **bcrypt** (Criptografia de senhas)
